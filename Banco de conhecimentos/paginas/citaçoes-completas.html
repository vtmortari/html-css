Criando TeCitações Completas

Podemos criar uma Citação Completas utilizando a tag <blockquote>.

Citações completas são utilizadas quando queremos referenciar grandes autores ou nomes famosos assim como providenciar para mecanismos de busca as fontes usadas nessa citação.

Em seu livro "Os Fundamentos do HTML e CSS", Maurício Samy diz:

HTML é a sigla em inglês para Hyper Text Markup Language.

Quando desejamos fornecer a fonte da citação usada, utilizamos o parâmetro "cite=" e anexamos o link. É importante ressaltar que isso não aparecerá no texto. É somente usado pelos mecanismos de busca.

Exemplo de Código 

            
                <p>
                Em seu livro "Os Fundamentos do HTML e CSS", Maurício Samy diz:
                 <blockquote cite="fonte">HTML é a sigla em inglês para Hyper Text Markup Language.</blockquote>
                </p>
