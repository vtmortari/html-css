Marcações de Texto

Utilizando Marcações

Para realizarmos marcações de texto,, utilizamos a tag <mark>.

Customizando Marcações

Podemos customizar nossas marcações de duas formas diferentes.
Podemos utilizar o parâmetro "style=" dentro da tag quando desejamos
alterar marcações específicas.

Quando desejamos alterar todas as marcações de uma vez, utilizamos o
seletor style. Utilizando das CSS para customizar nossas marcações.

Exemplo de Código 

            
                <p>
                    Exemplo <mark style="background-color:red">texto marcado</mark>
                </p>
