Listas Não Ordenadas

Quando desejamos criar uma lista onde a ordem dos elementos presentes não importa, como por exemplo, listas de mercado, utilizamos a tag <ul> (unordered list).

Sempre que criamos uma lista, para identificar os elementos dentro da tag como itens da lista, utilizamos a tag <li>.

Exemplo

  • Batata
  • Ovo
  • Leite

Exemplo de Código 

            
                <p>
                    <ul>
                        <li>Batata</li>
                        <li>Ovo</li>
                        <li>Leite</li>
                    </ul>
                </p>

Modificações

Podemos alterar qual o tipo de icone utilizado nos elementos da lista utilizando o parâmetro "type".

  • type="disc" - Para um disco preto pintado.
  • type="circle" - Para um circulo preto.
  • type="square" - Para um quadrado preto pintado.

Exemplo

  • Batata
  • Ovo
  • Leite

Exemplo de Código 

            
                <p>
                    <ul type="circle">
                        <li>Batata</li>
                        <li>Ovo</li>
                        <li>Leite</li>
                    </ul>
                </p>
