Listas Ordenadas

Quando desejamos criar uma lista onde a ordem dos elementos presentes nela são importantes, como por exemplo, listas de colocados, rankings ou qualquer coisa do tipo, utilizamos a tag <ol> (ordered list).

Sempre que criamos uma lista, para identificar os elementos dentro da tag como itens da lista, utilizamos a tag <li>.

Exemplo

  1. Primeiro
  2. Segundo
  3. Terceiro

Exemplo de Código 

            
                <p>
                    <ol>
                        <li>Primeiro</li>
                        <li>Segundo</li>
                        <li>Terceiro</li>
                    </ol>
                </p>

Modificações

Podemos alterar alguns elementos da lista, como o tipo de contagem feito ou de que ponto a contagem começa utilizando os seguintes parâmetros:

  • type: Nos permite mudar o tipo de contagem da lista.
    • type="1" - Para listas numéricas
    • type="A" - Para listas alfabéticas maiúsculas.
    • type="a" - Para listas alfabéticas minúsculas.
    • type="I" - Para listas romanas maiúsculas.
    • type="i" - Para listas romanas minúsculas.
  • Start: Nos permite escolher de que ponto da contagem nossa lista se inicia. Deve sempre ter valores numéricos.

Aplicando os Parâmetros

Exemplo

  1. Primeiro
  2. Segundo
  3. Terceiro

Exemplo de Código 

            
                <p>
                    <ol type="I" start="3">
                        <li>Primeiro</li>
                        <li>Segundo</li>
                        <li>Terceiro</li>
                    </ol>
                </p>
