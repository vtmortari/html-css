Hyperlinks

Hyperlinks são um recurso muito importante para o funcionamento e recomendação de nossos sites. Com eles podemos criar navegações internas e externas para nosso site, assim como criar links de download.

Por que tão importante?

Uma boa estruturação de hyperlinks é vital para uma boa expeciência de usuario e também para os algoritimos de recomendação de ferramentas de busca como o Google.

Criando um Hyperlink

Para criarmos um Hyperlink, utilizamos a tag <a>. Nela, preenchemos o parâmetro "href" com o URL do destino de nosso link.

Exemplo

Visite o YouTube

Exemplo de Código 

            
                <p>
                    <a href="https://www.youtube.com/">Visite o YouTube</a>
                </p>

Parâmetros

Quando criamos um hyperlink existem agluns parâmetros que são muito importantes para garantir uma boa funcionalidade.

  • hreflang - Define o idioma da página para qual o hyperlink leva.
  • target - Define como será aberta a aba do link.
    • target="_self" - Fará o link ser aberto na aba atual.
    • target="_blank" - Fará o link ser aberto em uma nova aba.
  • rel - Define a relação do link.
    • rel="next" - Indica que o link leva a proxima página do site atual.
    • rel="prev" - Indica que o link leva a uma página anterior do site atual.
    • rel="author" - Indica que o link leva a página do autor do link.
    • rel="external" - Indica que o link leva a uma página exerterna do site atual.
    • rel="nofollow" - Indica que o link é para um site não endosado. Dessa forma algoritimos de busca não seguirão esse link.
  • download - Colocamos a URL do arquivo que sera baixado.
  • type - Colocamos o midia type do arquivo a ser baixado.

Tipos de Hyperlink

Quando criamos nossos Hyperlinks, eles podem ser links Internos, Externos e de Download

Links Internos

Links internos representam navegações internas no nosso próprio site, links que levam de uma página a outra.

Exemplo

Volte a página principal

Exemplo de código 

            
                <p>
                     <a href="../Principal.html" target="_self" rel="prev">
                        Volte a página principal.
                     </a>
                </p>

Links Externos

Links externos são links que nos levam para páginas externas a nosso site.

Exemplo

Visite o YouTube

Exemplo de código 

            
                <p>
                     <a href="https://www.youtube.com/" target="_blank" rel="external">
                        Visite o YouTube
                     </a>
                </p>

Links de Download

São hyperlinks que nos permitem baixar arquivos hospedados no site.

Exemplo

Baixe meu favicon

Exemplo de Código 

            
                <p>
                    <a href="../imagens/favicon.ico" download="../imagens/favicon.ico" type="image/x-icon">
                        Baixe meu favicon
                    </a>
                </p>
