Listas de Definição

Quando queremos criar uma lista de definições como, por exemplo, um dicionário, utilizamos a tag <dl> (definition list).

Para identificarmos os termos de definição da lista, utlizamos a tag <dt> (definition term).

Para identificarmos a descrição do termo, utilizamos a tag <dd> (definition description).

Exemplo

HTML
Linguagem de marcação utilizada para criar os conteúdos de um site.

Exemplo de Código 

            
                <p>
                    <dl>
                        <dt>
                             HTML
                        </dt>
                        <dd>
                            Linguagem de marcação utilizada para criar os conteúdos de um site.
                        </dd>
                    </dl>
                </p>
