Imagens Dinâmicas

Quando queremos que o tamanho de nossa imagens se ajuste ao tamanho de tela disponivel do dispositivo que acessa nosso site, criamos uma imagem dinâmica.

Para criar uma imagem dinâmica, utilizamos a tag <picture> e dentro dela criamos uma tag <img> com nossa imagem final e acima criamos tags <source:midia:type> com as imagens alternativas que ajustaremos para cada tamanho disponível, assim como seu midia type e o tamanho máximo para uqal aquela imagem deve ser exibida.

A ordem dos elementos sempre deve ir da menor foto para a maior, terminando na foto original na tag <img>.

Exemplo

Tente mudar o tamanho da aba.

Exemplo de Código 

            
                <p>
                    Tente mudar o tamanho da aba. 

                    <picture>
                        <source media="(max-width: 750px )" srcset="../imagens/foto-p.png" type="image/png">
                        <source media="(max-width: 1050px )" srcset="../imagens/foto-m.png" type="image/png">
                        <img src="../imagens/foto-g.png" alt="foto grande">
                    </picture>
                </p>
